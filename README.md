## introduction

deno is a runtime similar to nodejs but it's built in `rust` and so is a lot faster

## getting started

install via

```js
choco install deno
```

create `helloWorld.ts`

```js
console.log('welcome to Deno runtime');
```

run via

```js
deno run helloWorld.ts
/*
welcome to Deno runtime
*/
```
